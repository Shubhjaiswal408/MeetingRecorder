Flash Firmware

Connect your XIAO ESP32-S3 Sense via USB and click the button below. Chrome or Edge is required.


Watch Tutorial

Supported Hardware

  • XIAO ESP32-S3 Sense
  • ESP32-S3 Chip Family
  • USB Data Cable Required

Installation Steps

  1. Connect ESP32-S3 via USB.
  2. Open Chrome or Edge.
  3. Click Flash Firmware.
  4. Select Serial Port.
  5. Wait until flashing completes.
  6. Board will reboot automatically.

Troubleshooting

  • Use a USB DATA cable.
  • Try another USB port.
  • Close Arduino IDE.
  • Disconnect serial monitors.
  • Enter bootloader mode manually.
  • Refresh browser and retry.
